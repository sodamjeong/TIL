팀 버너스리

프로그래머

1989년 월드 와이드 웹의 하이퍼텍스트 시스템을 고안하여 개발. 웹의 아버지

국적 : 영국

생년월일 : 1955년 6월 8일

추가 정보 : https://ko.wikipedia.org/wiki/팀_버너스리

생애

버너스리는 잉글랜드 런던에서 태어났으며,
부모는 메리 리 우즈(Mary Lee Woods)와 콘웨이 버너스리(Conway Berners-Lee)이다.
1990년에 낸시 칼슨(Nancy Carlson)과 결혼하였으나 2011년 이혼하였다.
2014년, 로즈메리 리스(Rosemary Leith)와 런던 세인트 제임스 궁전에서 결혼하였다.
